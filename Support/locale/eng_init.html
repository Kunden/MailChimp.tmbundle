Getting Started with the MailChimp Bundle

Create and Edit Templates

There are two ways you can work with a Template. Choose Bundles->MailChimp->New Template from Current File or Bundles->MailChimp->Get Template. If you’re creating a new Template, you’ll be prompted to enter a name. With “Get Template”, you’ll be presented with a list of Your Templates.

Working with Campaigns

Work with any Campaign that has been created using “Paste in Code”,”Import HTML” or using a Custom Template. Campaigns associated with a MailChimp Template can not be edited via the TextMate bundle.

Status Window

This window will be presented with feedback on most bundle actions.
